    Analyzer
    components/analyzer.js, line 28

    General analyzer component for composing interesting analyzes from a MonkeySet

    See:
    • https://tulipindicators.org/list
    Example
    // See this list for all analyzers you can use: https://tulipindicators.org/list
const rsi = await monkeyset.Analyzer.rsi({
  options: {
    period: 5
  },
  inputs: {
    real: monkeyset.Filter.get('column', 'close').last(10).end()
  }
})
